# Entregaveis do projeto

## Conteudo esperado

- `versao_final`: copia identificada da versao final do projeto.
- `instalador`: APK release para instalacao manual.
- `executavel`: referencia ao APK Android.
- `apk_aab_release`: APK e AAB assinados.
- `materiais_identificacao`: icone, splash, logotipos e capturas de tela.
- `link de acesso.txt`: links publicos do projeto.

## Importante

Os arquivos APK e AAB nao devem ser substituidos por arquivos vazios. Eles
precisam ser gerados pelo EAS Build, baixados e testados antes da entrega.
