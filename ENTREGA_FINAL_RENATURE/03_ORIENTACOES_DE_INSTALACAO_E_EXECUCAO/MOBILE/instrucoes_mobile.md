# Instrucoes mobile

## Instalar para desenvolvimento

1. Instale Node.js e npm.
2. Entre em `renature-mobile-project-main`.
3. Execute `npm install`.
4. Crie `.env` com `EXPO_PUBLIC_API_URL`.
5. Execute `npx expo start --clear`.
6. Abra o QR Code no Expo Go.

## Instalar o APK release

1. Baixe o arquivo `renature-v1.0.0-release.apk`.
2. No Android, permita a instalacao de aplicativos da fonte utilizada.
3. Abra o arquivo APK.
4. Confirme a instalacao.
5. Abra o aplicativo Renature.

## Fluxo basico de uso

1. Passe pelas telas introdutorias.
2. Selecione `Cadastre-se`.
3. Informe nome, e-mail e senha com pelo menos oito caracteres.
4. Efetue login.
5. Utilize a tela inicial para acessar scanner, registro manual, guias e
   conquistas.
6. Consulte o perfil para acessar configuracoes e creditos.
7. Em caso de esquecimento da senha, solicite o codigo por e-mail.

## Requisitos

- Android com acesso a Internet.
- Permissao de camera para o scanner.
- E-mail valido para cadastro e recuperacao de senha.
